Wat is Code Retreat?

Code Retreat is een intensieve workshop met een focus op de fundamenten van software development en design. Jij als developer krijgt de kans om gericht je vaardigheden te trainen, weg van de dagelijkse druk en chaos van "dingen af krijgen".

Het Code Retreat-format heeft zich bewezen als een zeer effectieve manier om je kennis en vaardigheden te verbeteren. Door de basis principes van test-driven, modulair en objectgeoriënteerd design te oefenen verbeter je je vaardigheden om code te schrijven die de cost-of-change-over-time minimaliseert.

Code Retreat is niet gebonden aan specifieke programmeertaal, omgeving of platform; gebruik wat je elke dag gebruikt. Door pair-programming kun je ook kennis maken met andere developers, manieren van werken, tooling, programmeertalen en test-frameworks.

Is CodeRetreat geschikt voor mij?

CodeRetreat gaat over leren van en met elkaar. Tijdens workshops zijn deelnemers van verschillende technische achtergronden en niveaus van harte welkom.

Het belangrijkste wat je mee moet nemen is een laptop met daarop jouw favoriete programmeertaal en testframework klaar voor gebruik. Je ontvangt voorafgaand aan de workshop meer informatie.

Code Retreat op locatie

Komt geen van de geplande workshops jou en je collega's goed uit? Of wil je gewoon met een vertrouwde groep mensen aan de slag? Geen probleem! Het is mogelijk de Code Retreat workshop op locatie te verzorgen.
