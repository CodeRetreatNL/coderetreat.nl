Wat is Code Retreat?

Code Retreat is een intensieve workshop met een focus op de fundamenten van software development en design. Jij als developer krijgt de kans om gericht je vaardigheden te trainen, weg van de dagelijkse druk en chaos van "dingen af krijgen".

Het Code Retreat-format heeft zich bewezen als een zeer effectieve manier om je kennis en vaardigheden te verbeteren. Door de basis principes van modulair en object-geörienteerd design te oefenen verbeter je je vaardigheden om code te schrijven die de _cost of change over time_ minimaliseert.

Code Retreat is niet gebonden aan specifieke programmeertaal, omgeving of platform; gebruik wat je elke dag gebruikt. Door pair-programming kun je ook kennis maken met andere developers, manieren van werken, tooling, programmeertalen en test-frameworks.
